# Watermark Logo

Bu papkada watermark faylini qo'ying.

**Qo'llab-quvvatlanadigan formatlar:**
- `watermark.png` (tavsiya etiladi - shaffof fon bilan)
- `watermark.jpg` yoki `watermark.jpeg`

**Talablar:**
- O'lcham: 200x60px yoki shunga o'xshash nisbat
- Sifat: Yuqori sifatli
- Format: PNG (shaffof fon bilan) yoki JPG

**Eslatma:**
- PNG formatda opacity (shaffoflik) qo'llanadi
- JPG formatda opacity qo'llanmaydi (to'liq ko'rinadi)

Watermark avtomatik ravishda barcha yuklangan rasmlarga qo'shiladi (o'ng pastki burchakda).
